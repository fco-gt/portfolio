# Portfolio

## Mi portafolio web creado con Nextjs

## Tecnologias usadas:
- NextJS
- NextUI
- Frame Motion
- React
- Tailwind CSS
- Tailwind Variants
- Next Themes
- Typescript/Tsx
- Javascript/Jsx

### Instrucciones de uso

Instalar dependencias

```bash
npm install
```

Ejecutar en modo desarrolador mediante

```bash
npm run dev
```

Abrir http://localhost:3000 para ver la web en tu navegador.
